# CDAC Course Management & Projects

Welcome to my CDAC (Centre for Development of Advanced Computing) repository. This repository serves as a centralized hub to maintain, track, and organize my entire academic curriculum, daily course content, lab assignments, and project development files.

---

## 📌 Purpose of this Repository
* **Coursework Tracking:** Maintaining daily lecture codes, notes, and lab exercises across various modules.
* **Project Management:** Hosting source code, documentation, and assets for academic and capstone projects.
* **Version Control:** Keeping a clean history of progress and code modifications throughout the course duration.

---
